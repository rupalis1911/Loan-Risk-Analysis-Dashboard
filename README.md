Loan Risk Analysis Dashboard (SQL + Power BI)

Problem Statement
Financial institutions need to identify high-risk applicants to reduce loan defaults and improve approval decisions.

 Tech Stack
- SQL (Data cleaning & transformation)
- Power BI (Dashboard & visualization)
- DAX (KPIs & calculations)

 Process
1. Cleaned and transformed raw loan data using SQL
2. Created derived columns like risk category & income bands
3. Imported the cleaned dataset into Power BI
4. Built an interactive dashboard with filters and KPIs

 Dashboard Features
- Total Applicants, Loan Amount, Approval Rate
- Risk segmentation (High / Medium / Low)
- Credit Score vs Approval analysis
- Income & Employment-based insights

 Key Insights
- High-risk applicants contribute significantly to total loan exposure
- Approval rate decreases with higher debt exposure
- Stable income applicants show lower risk
- Poor credit score leads to lower approval rates

 Business Impact
This dashboard helps identify high-risk applicants and improve loan approval strategies.
