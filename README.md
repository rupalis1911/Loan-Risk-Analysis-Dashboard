Loan Risk Analysis Dashboard (SQL + Power BI)

 Overview
This project analyzes loan application data using SQL and Power BI to identify high-risk applicants and improve loan approval.

 Key Insights
- High-risk applicants contribute significantly to total loan exposure
- Approval rate decreases with higher debt exposure
- Stable income applicants show lower risk
- Poor credit scores have the lowest approval rates

 Features
- KPI cards: Total Applicants, Loan Amount, Approval Rate
- Risk segmentation analysis
- Credit score vs approval heatmap
- Interactive filters (income, employment, age)

 Process
1. Used SQL to clean and prepare the raw loan dataset
2. Created derived columns (risk category, income bands, etc.)
3. Imported cleaned data into Power BI
4. Built an interactive dashboard with KPIs and filters

 Tools Used
- SQL
- Power BI
- DAX
- Data Visualization

 Outcome
This dashboard helps in making data-driven lending decisions and reducing loan default risk.
